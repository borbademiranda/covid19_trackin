Covid-19 no Brasil

O total de casos registrados no Brasil até o dia 20/11/2020 às 17:13:12 é de 5981767. Podemos observar no mapa abaixo algumas estatísticas sobre a Covid-19 nos municípios brasileiros. Os dados são atualizados diariamente, dependendo das secretarias estaduais de saúde, tendo em vista que os dados são produzidos a nível municipal. Os dados são provenientes do site Brasil.io e da Johns Hopkins University.
