O total de casos registrados no Brasil até o dia 18/04/2020 às 22:35:20 é de 30425. Podemos observar no mapa abaixo algumas estatísticas sobre a Covid-19 nos municípios brasileiros. Os dados são atualizados diariamente, dependendo das secretarias estaduais de saúde, tendo em vista que os dados são produzidos a nível municipal.
